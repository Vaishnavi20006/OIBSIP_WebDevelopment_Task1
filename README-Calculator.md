# Calculator Project

## Objective
To create a basic calculator using HTML, CSS, and JavaScript that can perform simple mathematical operations.

## Steps Performed
1. Created the structure of the calculator using HTML.
2. Styled the calculator buttons and layout using CSS Grid.
3. Added functionality for addition, subtraction, multiplication, and division using JavaScript.
4. Tested the calculator to ensure all functions work properly.

## Tools Used
- HTML
- CSS
- JavaScript
- VS Code

## Outcome
A working calculator web application that takes user input and performs basic arithmetic operations accurately.
